# `spikes/`

Throwaway experiments. Code here answers one specific unknown so the real implementation is informed by reality, not assumptions. Nothing in this directory ships — the knowledge produced lives in `docs/research/` as a written characterization.

Spikes are `.gitignore`'d? No — we commit them so the experiment can be re-run later if the external service changes behavior. But the code is marked "do not import from packages/" and is not part of `uv sync`.
